## 短網址產生

將較長的網址縮短成短網址

## 開始使用

1. 確認點腦是否有node.js與npm

2. 將專案複製到主機上

```
git clone 
```


3. 將相關模塊下載至本機上

```
$ npm install
```

4. 啟動伺服器

```
$ npm run dev
```

5. 如果伺服器成功啟動，會顯示此提示

```
App is running on http://localhost:3000
```
